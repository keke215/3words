# あなたの人生に最も大切なモノジェネレータ


## 概要

	この中で最初に見つけた３つのワードが人生に最も大切な言葉です。

の結果を、絶対に隣接する漢字で単語ができないような組み合わせにしようぜ


## 使い方

	ruby main.rb

パラメータ

	length: 表の大きさ
	words:  漢字の種類

## 漢字リスト

-  kanji: 常用漢字
-  kanji_easy: 小学生の習う漢字
-  kanji_easy2: 小学生の習う漢字から非、無などを除いたやつ


## 結果の目安

常用漢字だったら４０マスでも行ける


## 辞書

- IPA辞書
- unidic(国立国語研究所)

unidicの追加で精度が向上したけど、めっちゃ重くなった模様。

辞書の最適化をすればもっと軽くなるかと。
